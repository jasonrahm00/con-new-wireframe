Mega Menu and Mobile Compatibility

  • There need to be main landing pages for all of the top-leve nav items, but we also need a to click and expose submenus on mobile (maybe on desktop to, for accessibility). Look at Regis University mobile view for inspiration.
  • We could combine the Admissions and Programs under one mega, mega menu and use the main nav component as a click target to expose the menu. Admissions and Programs will become their own headers that are clickable to the page, as on the main UC Denver page, but I don't like this option. Menu will become too jumbled too fast.
  • Another option is to eliminate sub-menus altogether and drop them onto landing pages. This could be a down-the-road option as the programs and admissions pages will need some custom sorting to get users to the right page. Look at main CU Boulder site for inspiration.
  • Worcester Polytechnic uses an expand button that toggles the submenu while still making the main page/header a clickable link. Might not be the best solution, but it is a solution. Separate toggle button and link with a vertical line.
  • Saint Joseph's University has an additional "about" button for each nav page that shows up in mobile and desktop that will take you to the landing page.

Apply Now Link

There is an Apply Now page on the site that could be the destination of the 'apply now' button. The page and links will need to be redone. Ideally it'll target the specific application page for that program.

We will need some sort of "application process" or "how to apply" page for each degree level, so they can come prepared with necessary documents.
